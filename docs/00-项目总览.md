# SillyChat - 项目总览

## 项目概述

**项目名称**: SillyChat
**基于**: OpenClaw 深度定制
**目标平台**: Windows、macOS、Android
**定位**: AI原生聊天工具，个人和团队AI协作平台

## 核心特性

1. **AI优先架构** - 从底层为AI交互设计优化
2. **本地化存储** - 聊天记录本地存储，隐私可控
3. **文件自动整理** - 图片、视频、文档自动分类，去重存储
4. **多AI管理** - 每个账号可配置多个AI代理，自由切换角色
5. **真人+AI分身** - 双重身份，可在群组中邀请AI分身
6. **场景化聊天** - 项目、闲聊、会议等多场景支持
7. **Skills技能中心** - 与挺傻网站Skills同步，支持上传、出售、采购
8. **端到端加密** - 安全的通信保障
9. **云端同步** - 支持用户自有云端存储
10. **多语言UI** - 12种语言界面支持
11. **多语言实时翻译** - 三种显示模式，支持语言学习
12. **自愈反馈机制** - 自动bug收集与大模型自动修复
13. **生活服务与支付** - AI驱动的智能生活服务平台

## 技术栈

### 后端核心
- **Node.js** >= 22.12.0
- **TypeScript** >= 5.9
- **SQLite** + sqlite-vec
- **Zod** 数据验证

### 前端界面
- **Windows**: Electron + React/Vue
- **macOS**: SwiftUI + OpenClawKit
- **Android**: Jetpack Compose

### AI模型支持
- OpenClaw (Clawd) - 内置
- Anthropic Claude 3.5/4
- OpenAI GPT-4/GPT-4o
- Ollama 本地模型

## 关键设计决策

### 1. 为什么基于OpenClaw？
- 成熟的AI代理架构
- 多模型支持
- 技能系统完善
- 开源可定制

### 2. 为什么本地化优先？
- 用户隐私保护
- 降低服务器成本
- 离线可用
- 数据自主权

### 3. 为什么需要AI分身？
- 一个身份多种角色
- 场景化AI能力
- 群组协作需求
- 个性化体验
